# 南京医保家庭账户充值

::: tip 背景

由于不经常使用，随着年纪增长总会忘记，特记录下步骤，以备不时之需。

> 所有操作均在 "我的南京" APP 中完成。

<img src="https://cdn.jsdelivr.net/gh/jonaslgtm/fucang-imgs@latest/my-nanjing-ios.png" style="width:16%;">

:::

## 创建医保家庭账户（户主）

1. 打开 **我的南京** APP

2. 首页点击 **医保服务**

3. 当前页面点击 **医保家庭账户管理**

4. 点击 **创建家庭账户**  - 按系统指引操作

![南京医保家庭账户](https://cdn.jsdelivr.net/gh/jonaslgtm/fucang-imgs@latest/nj-yibao-01.avif)

5. 创建成功后，再回到 **家庭账户管理** 的首页，进入 **我是户主**

6. **绑定家人的账号** - 也按照系统指引操作，可以绑定多人

7. 点击 **个人账户注入**，按照指引将自己个人账户余额充到家庭账户中

![南京医保家庭账户](https://cdn.jsdelivr.net/gh/jonaslgtm/fucang-imgs@latest/nj-yibao-02.avif)

::: danger 需要注意的是：

一个主账户人只能申请一个家庭共济关系，且主账户人的账户余额只能单向流出，不能用家庭成员的账户余额。

:::

## 非户主充值指南（家庭成员）

作为家庭成员，想把医保账户余额充值到家庭账户中，按如下步骤操作即可。

1. 打开 **我的南京** APP

2. 首页点击 **医保服务**

3. 往下滑找到 **个账亲属转移**

4. 输入 相关信息（户主的姓名、身份证、转账金额）

![南京医保家庭账户](https://cdn.jsdelivr.net/gh/jonaslgtm/fucang-imgs@latest/nj-yibao-03.avif)

5. 确定后会弹出 个人承诺书，没有问题点击确定即可。

![南京医保家庭账户](https://cdn.jsdelivr.net/gh/jonaslgtm/fucang-imgs@latest/nj-yibao-04.avif)

💡 转账成功后，联系户主进行 **个人账户注入** ，即可完成个人账户余额充到家庭账户中。
